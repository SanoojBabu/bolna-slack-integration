# Bolna AI-Slack Integration

I have developed this project to work in local and production in support of **ngrok|cloudflared**.

I have utilized all the features given by both the platforms.

**Features used**

**Bolna.AI** - Agents->Analytics->Webhooks->Raw Data Send (Every related Documentaion)

**Slack** - Developer Apps->Webhooks->Open URL Provided->Triggered->Notifications Comes into the specified Channel (Every ralted Documentation)
